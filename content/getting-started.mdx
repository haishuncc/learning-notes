# 如何添加笔记

所有笔记都放在 `content` 目录中，支持 `.md` 和 `.mdx` 文件。

## 新增文章

例如创建 `content/javascript.md`：

```md
---
title: JavaScript 学习笔记
---

# JavaScript 学习笔记

这里开始写正文。
```

然后在 `content/_meta.js` 中把它加入目录：

```js
export default {
  index: '首页',
  javascript: 'JavaScript'
}
```

## 创建分类

可以使用文件夹整理同一类笔记：

```text
content/
├── frontend/
│   ├── _meta.js
│   ├── javascript.md
│   └── react.md
└── backend/
    ├── _meta.js
    └── nodejs.md
```

每个文件夹中的 `_meta.js` 用来设置该分类下文章的名称和顺序。
